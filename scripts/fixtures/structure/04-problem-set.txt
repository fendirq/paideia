Calculus I — Problem Set 7: Derivatives and Applications

Instructions: Show all work. Partial credit will be given for correct methodology even if the final answer is wrong. Each problem is worth 10 points.

Problem 1. Find the derivative of f(x) = 3x^4 - 5x^2 + 2x - 7 using the power rule.

Problem 2. Differentiate g(x) = (2x + 1)(x^2 - 3) using the product rule. Do NOT expand first.

Problem 3. Compute dy/dx for y = sin(x^2 + 1) using the chain rule.

Problem 4. Find the equation of the tangent line to f(x) = x^3 - 2x at the point x = 1.

Problem 5. A particle moves along a straight line with position s(t) = t^3 - 6t^2 + 9t (in meters, t in seconds, t ≥ 0).
  (a) Find the velocity function v(t).
  (b) At what time(s) is the particle at rest?
  (c) Find the total distance traveled over the interval [0, 4]. (Hint: watch the sign of v(t).)

Problem 6. A rectangular garden is to be built against a long wall, so only three sides need to be fenced. You have 40 meters of fencing. What dimensions maximize the enclosed area? Justify your answer using the first or second derivative test.
